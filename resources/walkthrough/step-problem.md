# Setup a Problem Instantly

Paste any problem URL and CP Studio will:

1. Create the directory structure
2. Fetch the problem statement
3. Download sample test cases
4. Generate a solution template

**Supported platforms:**
- Codeforces (`codeforces.com/contest/...`)
- LeetCode (`leetcode.com/problems/...`)
- GeeksforGeeks (`geeksforgeeks.org/problems/...`)

Try it with: `https://codeforces.com/contest/1/problem/A`
